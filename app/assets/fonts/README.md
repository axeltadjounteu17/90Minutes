# 90Minutes — Required Fonts

Download these fonts and place the .ttf files in this directory:

## Oswald (Display/Headlines)
- Download from: https://fonts.google.com/specimen/Oswald
- Files needed:
  - Oswald-Bold.ttf
  - Oswald-SemiBold.ttf

## Inter (Body text)
- Download from: https://fonts.google.com/specimen/Inter
- Files needed:
  - Inter-Regular.ttf
  - Inter-SemiBold.ttf
  - Inter-Bold.ttf

## JetBrains Mono (Scores/Stats)
- Download from: https://fonts.google.com/specimen/JetBrains+Mono
- Files needed:
  - JetBrainsMono-Bold.ttf

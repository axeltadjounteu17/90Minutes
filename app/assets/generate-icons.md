# 90Minutes — Icon Generation Guide

## Required files:
- `icon.png` — 1024x1024px (app icon for iOS/Android)
- `favicon.png` — 48x48px (web favicon)
- `adaptive-icon.png` — 1024x1024px (Android adaptive icon, with padding)
- `splash-icon.png` — 200x200px (splash screen icon)

## Quick generation:
1. Open icon.svg in a browser
2. Use https://realfavicongenerator.net or https://svgtopng.com
3. Export at 1024x1024 → save as icon.png
4. Resize to 48x48 → save as favicon.png

## Design specs:
- Background: #000000 (Adidas black)
- Football icon: white
- "90" text: #FF9900 (AWS orange)
- Bottom accent: #D0021B (Bundesliga red)
- Corner radius: 224px (iOS will clip automatically)
